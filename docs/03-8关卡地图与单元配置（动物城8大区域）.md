# 8关卡地图与单元配置（动物城8大区域）

目标：固定“8单元→8关卡”的叙事与配置格式，后续工程实现以配置驱动，不把文案散落在代码里。

## 1. 8大区域（你给定的世界观）
1. **🐸 水滴镇**（第一单元）- 小蝌蚪找妈妈主题
2. **🌳 森林区**（第二单元）- 树之歌、田家四季主题
3. **🌈 彩虹桥**（第三单元）- 星空、彩虹主题
4. **⛰️ 高山区**（第四单元）- 黄山、日月潭主题
5. **🎭 寓言街**（第五单元）- 坐井观天、寓言故事主题
6. **🏅 英雄广场**（第六单元）- 革命故事主题
7. **☁️ 云雾谷**（第七单元）- 江雪、雾的世界主题
8. **🎪 欢乐园**（第八单元）- 称赞、快乐主题

## 3. 单元节奏（难度与题型逐步引入）
- U1：重点识字/拼音（T1），少量 T2/T3（建立信心）
- U2：增加词语辨析（T2）与诗句/积累（T4 小比例）
- U3：句子仿写（T3）比例上升（更强调表达）
- U4：景点类课文理解（T5）增加（选择/判断为主）
- U5：寓言理解（T5）+ 易混词辨析（T2）
- U6：叙事类理解（T5）占比增加，题目更长但题量可减少
- U7：古诗背诵（T4）加大（填空/连线）
- U8：综合关（五题型混合），偏“快乐复盘”

## 4. 解锁与重复游玩
- MVP：默认 8 关全部可点（更适合复习自由度）
- v1：可加解锁：上一关 ≥⭐⭐ 解锁下一关（可选开关）
- 重复游玩：任何关都可无限挑战；每次 run 随机出题
